# Desafio do curso Ignite da Rocktseat
Consiste em criar uma lista de tarefas, de acordo com o que o usuário digita no input.

![](todo.gif)

## Funcionalidades:
  * Criar tarefa
  * Listar tarefas
  * Excluir tarefa
  * Concluir tarefa
  * Contador de tarefas concluídas e criadas
